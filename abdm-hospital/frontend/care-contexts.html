About Care Contexts

Care contexts represent specific episodes of care for a patient. Each context:

  • Links patient data to ABDM Gateway
  • Enables secure data sharing
  • Maintains audit trail
  • Supports consent-based access

Note: Patient must have ABHA ID for linking.
